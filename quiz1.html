Qualitative Analysis

With the purpose of doing a qualitative analysis of the images quality, we decided to create a quiz. This allowed us to make a cross reference between the system memory used vs the images that people chosen by answering the quiz. We used the ClipSize and the ClipRange as test parameters. Our intention was to approach people who were not familiar with the subject. Hence, the answers are more plausible. After gathering the opinion of thirty people, the analysis can be consulted below.

Do you consider yourself a gamer or somewhat aware of modern real time graphics?

The most voted choice in this question was the third image once more, because the first ones were too dark. The ClipSize was define at 256. So if the ClipRange is not big enough, it can not include the complete scene to voxelize. So these parts of the scene do not contribute to the global illumination and if not directly illuminated they are black as shown in the images. Question 6
Figure 16 Figure 17 Figure 18 Figure 19

Comparing this four images, the majory voted on the second image. The parameters used on this one are ClipSize at 128 and ClipRange at 512. The last one uses the ClipSize at 256 and a ClipRange at 1024. This last image uses 1843MB of graphics card memory and 2124MB of RAM while the most voted only uses 972MB of graphics card memory and 10MB of RAM. So, we can concluded that it is not necessary to use the best parameters possible to achieve a good image quality. Hence, we use less computational resources and have a better frame rate (55 frames per second).
